# Реферальные программы конкурентов: Cursor, Claude, Anthropic

Исследование на основе открытых источников (форум Cursor, Claude Help Center, профильные блоги). Данные собраны для проектирования реферальной программы Veai.

## Cursor

**Тип:** «Refer a friend» — постоянная реферальная программа со скидкой и кредитами.

- **Что получает приглашённый:** новые пользователи Cursor получают **50% скидку** на checkout в Stripe на первый платёж по тарифам Pro, Pro+ или Ultra.
- **Что получает реферрер:** после первой оплаты приглашённого реферрер получает **on-demand кредиты**.
  - Кредиты применяются **после исчерпания включённого в тариф объёма** использования.
  - Не передаются другим и не возвращаются деньгами (non-transferable, non-refundable).
  - На self-serve командах кредиты начисляются **лично пригласившему** (на его аккаунт, а не в общий командный пул).
- **Механика приглашения:** персональная реферальная ссылка или приглашение по email.
- **Начисление:** кредиты падают на баланс и «автоматически применяются к следующему инвойсу» — то есть фактически в следующем биллинг-цикле, а не мгновенно.
- **Болевые точки (из обсуждений на форуме):** непрозрачный UI — нет понятного раздела «Upcoming / Available Credit», путаница с тем, когда именно применяются кредиты; программа какое-то время воспринималась как «скрытая» / не для всех Pro-пользователей.

Источники: forum.cursor.com/t/cursor-referral-program/40555, devproblems.com, dredyson.com

## Claude (Anthropic) — «Claude 4 Invite Contest»

**Тип:** конкурс-розыгрыш (sweepstakes) за приглашения, ограниченный по времени.

- **Приз:** 4 месяца плана **Max** для приложений Claude (оценочная стоимость ~$400).
  - Текущие Max-пользователи: купон **−$100** на 4 месяца.
  - Max 5x: 4 месяца Max бесплатно.
  - Max 20x: купон −$100 на 4 месяца (доплата $100/мес за уровень 20x остаётся).
- **Механика:** уникальная реферальная ссылка в in-app баннере (нижний левый угол на desktop/mobile).
- **Условие зачёта entry:** новый пользователь должен (1) зарегистрироваться по ссылке и (2) отправить хотя бы одно сообщение Claude.
- **Накопление:** 1 entry за каждого реферала, копятся **10 дней** (срок конкурса), максимум **100 entry** на пользователя.
- **Логика:** больше успешных рефералов → больше шансов в розыгрыше.

Источники: support.claude.com/en/articles/11408405, support.claude.com/en/articles/11140763 (Official Rules)

## Claude Code `/passes` (Anthropic) — гостевые пропуска

**Тип:** реферальная фича для разработчиков, встроенная в CLI (с декабря 2025, вместе с Opus 4.5).

- **Кому доступно:** подписчикам тарифа **Max** ($100 или $200/мес).
- **Что раздают:** **3 одноразовых пропуска**, каждый даёт другу **7-дневный триал Claude Pro** без оплаты вперёд.
- **Что получает отправитель:** за каждого друга, который после триала **перешёл на платный Pro**, отправитель получает **$10 кредита Claude + "extra usage"**.
  - Все 3 конверсии = **$30 кредитов** (примерно месяц Pro).
- **Механика:** команда `/passes` только в Claude Code CLI (не в Desktop/web); генерирует ссылку вида `https://claude.ai/referral/{code}`.
  - Ссылка одноразовая, самопогашение заблокировано (нельзя активировать свой же пропуск).
  - Пополнения пропусков нет — только 3 на аккаунт.
- **Стратегия Anthropic:** таргетирование на разработчиков «живущих в терминале» → расчёт, что они приведут других разработчиков (точечный growth вместо массового промо).

Источники: thekodelab.com/en/posts/claude-code-passes, help.apiyi.com, opentherank.com, startupspells.com

## Прочее у Anthropic

- **Usage Bundles giveaway (апрель 2026):** разовая раздача free usage credit подписчикам активных премиум-планов, до **$200**, в честь запуска гибкого биллинга «Usage Bundles». Это не реферальная программа, но релевантный приём вовлечения. Источник: elegantaccess.org

## Выводы для Veai

1. **Cursor** = классическая модель «скидка другу + кредиты рефереру»; слабое место — непрозрачность начисления. Veai может выиграть на понятном UI статуса кредитов.
2. **Claude Contest** = геймификация через розыгрыш, низкий порог зачёта (1 сообщение) → широкая воронка, но награда вероятностная.
3. **Claude `/passes`** = таргет на разработчиков прямо в инструменте (CLI), реальный денежный бонус за конверсию. Сильный приём для B2D-продукта вроде Veai: реферальную механику можно встроить прямо в IDE-агента.
