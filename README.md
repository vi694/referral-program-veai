# Реферальная программа Veai

Спецификация и мокапы двусторонней реферальной механики Veai: «Пригласи коллегу — бонус получаете оба после первой подписки».

## Что внутри

| Файл | Назначение |
|------|-----------|
| [`SPEC.md`](SPEC.md) | Полная спецификация механики: шаги, награды, защита от фрода, идеи на будущее |
| [`mockups/invite-friend.html`](mockups/invite-friend.html) | Мокап раздела «Пригласить друга»: реферальная ссылка, статистика, шаги |
| [`mockups/future-achievements.html`](mockups/future-achievements.html) | Мокап будущих фич: ачивки, минуты за отзывы, программа амбассадоров |

## Суть механики

Двусторонняя win-win: пригласи коллегу → вы оба получаете бонус, когда он оформит первую подписку.

- **Реферер:** +350 минут на баланс после оплаты другом первого месяца.
- **Реферал:** скидка 50% на первую подписку (любой тариф, скидка только на первый месяц).

Бонус начисляется за первую оплаченную подписку, а не за регистрацию — это базовая защита от фрода.

## Как открыть мокапы

Откройте HTML-файлы из `mockups/` в браузере — они самодостаточны, без внешних зависимостей.
